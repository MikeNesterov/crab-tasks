# Миграция хостинга crab-tasks: Исследование

*Дата: 2026-02-10*

## Текущий стек
- **Хостинг:** GitHub Pages (бесплатно)
- **URL:** https://mikenesterov.github.io/crab-tasks/
- **Тип сайта:** Полностью статический (HTML/CSS/JS + JSON)
- **Размер:** ~53 KB (без .git)
- **CI/CD:** Push to main → автодеплой

---

## Варианты

### 1. Cloudflare Pages ⭐ Рекомендация
| | |
|---|---|
| **Цена** | Бесплатно (unlimited bandwidth!) |
| **CDN** | Глобальный edge network Cloudflare (300+ городов) |
| **Деплой** | Git push → автобилд |
| **Custom domain** | Да, бесплатный SSL |
| **Приватность** | Работает с приватными репо |
| **Плюсы** | Самый быстрый для статики, unlimited bandwidth, Workers для будущего бэкенда |
| **Минусы** | Ещё один аккаунт/сервис |

### 2. Vercel
| | |
|---|---|
| **Цена** | Бесплатно (100 GB bandwidth/мес) |
| **CDN** | Edge network |
| **Деплой** | Git push → автобилд |
| **Custom domain** | Да, бесплатный SSL |
| **Плюсы** | Отличный DX, preview deploys на каждый PR |
| **Минусы** | Лимит bandwidth (horror stories с неожиданными счетами), overkill для статики |

### 3. Свой VPS (Coolify/Nginx)
| | |
|---|---|
| **Цена** | $5-10/мес (Hetzner/DigitalOcean) |
| **Контроль** | Полный |
| **Плюсы** | Можно разместить и другие сервисы, полный контроль |
| **Минусы** | Нужно поддерживать сервер, SSL сертификаты, мониторинг. Overkill для статического сайта |

### 4. Оставить GitHub Pages ✅
| | |
|---|---|
| **Цена** | Бесплатно |
| **CDN** | Fastly CDN |
| **Плюсы** | Уже работает, ничего не нужно делать, хорошая интеграция с репо |
| **Минусы** | Только публичные репо (или платный GitHub), нет серверной логики |

---

## Вывод

Для **чисто статического сайта** без серверной логики GitHub Pages — отличный выбор, и миграция не даст значительных преимуществ.

**Имеет смысл мигрировать если:**
- Нужен приватный репозиторий → **Cloudflare Pages**
- Нужен серверный рендеринг/API → **Vercel** или **Cloudflare Workers**
- Нужен полный контроль + другие сервисы → **VPS + Coolify**

**Рекомендация:** Оставить GitHub Pages пока не появится конкретная потребность. Если мигрировать — Cloudflare Pages (бесплатно, быстрее, unlimited bandwidth, приватные репо).
